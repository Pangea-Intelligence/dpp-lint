# Security Policy

## Reporting a vulnerability

Do not open a public GitHub issue for security vulnerabilities.

Report them by email to **contact@pangea-intelligence.eu**. Include a description of the issue, steps to reproduce, and the affected version if known. You will receive an acknowledgement within 7 days.

Please allow a reasonable window for a fix before any public disclosure.

## Supported versions

Only the latest 0.x release receives security fixes. Older releases are not patched; upgrade to the current version.

## Scope

dpp-lint is a local CLI. It reads the files you point it at and sends no data anywhere. Outdated bundled data snapshots (CAHRA, RMI) are maintenance issues, not vulnerabilities; use a regular issue for those.
